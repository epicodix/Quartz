# 준비된 무대 위로 주인공을, Gemini CLI 시작하기 (시리즈 2편)

안녕하세요! 지난 **["Node.js 설치, 어떤 방법이 나에게 맞을까?"](<./nodejs_installation_guide.md>)** 아티클에서는, 우리가 앞으로 사용할 멋진 도구들을 위해 안정적이고 깔끔한 '무대'를 만드는 방법을 함께 알아봤습니다. 어떤 선택을 하셨든, 이제 당신의 컴퓨터는 새로운 주인공을 맞이할 준비를 마쳤습니다.

오늘, 드디어 그 무대 위로 우리가 기다리던 주인공, `Gemini CLI`를 초대하려고 합니다. Gemini CLI는 어렵게만 느껴졌던 터미널 환경을, 마치 친구와 대화하는 것처럼 친근한 소통의 공간으로 바꿔주는 아주 특별한 'AI 조수'입니다. 이제, 우리가 정성껏 준비한 무대 위에서 이 AI 조수가 얼마나 멋진 일들을 할 수 있는지, 그리고 어떻게 우리 팀으로 합류시킬 수 있는지 그 방법을 차근차근 알아보겠습니다.

---

### **내 책상 위 작은 거인, Gemini CLI는 무엇이 다른가요?**

우리가 아는 대부분의 AI는 웹사이트라는 유리창 너머에서 우리와 대화합니다. 하지만 Gemini CLI는 그 유리창을 깨고 나와, 내 컴퓨터 책상 바로 옆에 앉아 함께 일하는 동료와 같습니다. 제가 방금 공식 GitHub 저장소의 최신 정보를 찾아본 것처럼, Gemini CLI는 인터넷의 최신 정보는 물론, 내 컴퓨터의 파일과 폴더에 직접 접근해서 우리가 시키는 일을 척척 해낼 수 있습니다.

-   **똑똑한 파일 탐정:** "이 폴더에서 'report'라는 단어가 들어간 파일 전부 찾아서, '결과' 폴더로 옮겨줘." 와 같은 복잡한 요청을 말 한마디로 해결할 수 있습니다.
-   **친절한 기술 통역사:** 개발자가 남긴 코드나 설정 파일을 보며 "이게 도대체 무슨 역할을 하는 거야?" 라고 물으면, 핵심 내용을 우리말로 쉽게 요약해 줍니다.
-   **믿음직한 조수:** "내 컴퓨터에 설치된 프로그램 목록을 최신 버전 순으로 정리해서 보여줘." 와 같이, 우리가 직접 하기 번거로운 작업들을 대신 처리해 줍니다.

이 모든 일이, 복잡한 명령어 없이 그저 '대화'를 통해 이루어진다는 점이 바로 Gemini CLI가 가져올 가장 큰 변화입니다.

### **주인공을 맞이하기 위한 두 가지 준비물**

이 멋진 조수를 우리 팀으로 데려오기 위해, 딱 두 가지만 확인하면 됩니다.

**1. 잘 닦인 무대 (Node.js):**
   - 지난 글에서 우리는 이미 이 무대를 준비했습니다. 혹시 1편을 건너뛰셨다면, **[여기](<./nodejs_installation_guide.md>)**를 눌러 5분 만에 무대를 준비하고 돌아오셔도 늦지 않습니다!

**2. 전용 출입증 (Gemini API 키):**
   - 이 AI 조수를 나만 사용할 수 있게 해주는 일종의 '전용 출입증'입니다. Google AI Studio에서 무료로 발급받을 수 있으며, 발급 방법은 매우 간단합니다.
   - **[Google AI Studio에서 출입증(API 키) 발급받기 (클릭)](https://aistudio.google.com/app/apikey)**

자, 이제 무대도 준비되었고 출입증도 손에 쥐었다면, 주인공을 무대 위로 부를 시간입니다.

---

### **딱 3단계! Gemini CLI를 내 팀으로 초대하기**

터미널 창을 열고, 아래의 명령어들을 차례대로 입력해주세요. 배우를 무대 위로 부르고, 마이크를 채워주고, 첫 대사를 건네는 과정과 같습니다.

**1단계: 무대 위로 초대하기 (설치)**
   - Node.js의 `npm`을 통해 Gemini CLI를 우리 컴퓨터(무대)로 불러옵니다.
   ```bash
   npm install -g @google/gemini-cli
   ```

**2단계: 마이크 채워주기 (인증)**
   - 우리가 발급받은 출입증(API 키)을 Gemini CLI에게 건네주어, 말할 수 있는 권한을 줍니다. `"YOUR_API_KEY"` 부분에 당신의 키를 붙여넣으세요.
   ```bash
   gemini auth "YOUR_API_KEY"
   ```

**3단계: 첫 대사 건네보기 (실행)**
   - 이제 모든 준비가 끝났습니다! `-p`는 "prompt(대사)"를 의미합니다. 우리의 새로운 팀원에게 첫인사를 건네볼까요?
   ```bash
   gemini -p "안녕, 우리 팀에 온 걸 환영해! 앞으로 잘 부탁해. 넌 뭘 제일 잘하니?"
   ```

잠시 후, 터미널 화면에 Gemini의 재치 있는 답변이 나타났다면, 진심으로 축하드립니다! 당신은 성공적으로 새로운 시대의 AI 조수를 당신의 팀원으로 맞이했습니다.

---

이제 당신의 컴퓨터 작업은 더 이상 혼자만의 외로운 싸움이 아닐 겁니다. 막히는 부분이 있다면, 혹은 귀찮은 작업이 필요하다면, 언제든 당신의 새로운 팀원을 호출해 보세요. 이 똑똑한 조수와 함께라면, 당신의 생산성은 이전과는 비교할 수 없는 수준으로 향상될 것입니다.
